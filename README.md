# 🚀 Alien Invasion - 太空射击游戏

## 📌 游戏简介

Alien Invasion 是一款基于 Pygame 开发的经典太空射击游戏。玩家控制一艘宇宙飞船，在太空中与外星人舰队作战，通过精准射击和灵活躲避来获得高分。

## 🎮 游戏特性

### 🎮 核心玩法

多关卡系统 包含4个不同难度的关卡，每个关卡有独特的外星人阵型

护盾系统 飞船配备可再生的能量护盾，抵挡敌方攻击

粒子效果 炫酷的爆炸、火花和护盾受损特效

屏幕抖动 受到攻击时产生震撼的屏幕震动效果
### 🎯 游戏特色

智能敌人 外星人会根据位置智能射击，增加游戏挑战性

动态难度 随着关卡推进，游戏速度逐渐提升

多种武器 玩家子弹与外星人子弹各有特色

生命系统 三条生命机会，护盾耗尽后飞船会被摧毁
### ⚙️ 自定义设置

难度选择 简单、普通、困难三种难度模式

音量控制 独立的音乐和音效音量调节

视觉选项 可开关FPS显示

多语言支持 自动检测系统字体，支持中英文界面

## 💻 系统要求

### 最低配置

操作系统 Windows 7/10/11, macOS 10.12+, Linux

Python版本 Python 3.7 或更高版本

内存 512 MB RAM

存储空间 50 MB 可用空间
### 推荐配置

操作系统 Windows 10/11, macOS 11+, Ubuntu 18.04+

Python版本 Python 3.9+

内存 1 GB RAM

显卡 支持OpenGL 2.0

### 📥 安装指南

1. 安装Python
   python --version  # 检查Python版本
   

2. 安装Pygame库
   pip install pygame
   

3. 下载游戏文件
   git clone https://github.com/wzf2077/alien-invasion.git
   cd alien-invasion
   

4. 运行游戏
   python alien_invasion.py
   

## 🕹️ 游戏控制

### 基本控制

← → / A D 左右移动飞船

空格键 发射子弹

P键 暂停/继续游戏

ESC键 打开暂停菜单

R键 重新开始游戏

Q键 退出游戏
### 菜单导航

方向键/W S键 上下选择选项

回车键/空格键 确认选择

A D键/左右方向键 调整设置数值

## 游戏界面说明

### 主界面元素

分数显示 右上角 显示当前得分

最高分 屏幕顶部中央 显示历史最高分

等级显示 分数下方 显示当前关卡

生命指示 左上角 显示剩余飞船数量

护盾条 飞船上方 显示护盾能量值

## 🌌 关卡系统

### 关卡特色

第1关 3×3方阵 9个

第2关 金字塔形阵型 18个

第3关 菱形排列 27个

第4关 双线波浪形 36个
### 难度递增

外星人移动速度 逐关提升

外星人射击频率 逐关增加

得分倍数 随关卡提高

## ⭐ 得分规则

### 基础得分

击落普通外星人 50分

难度提升后 相应增加
### 连击奖励

连续击落外星人 额外分数加成

完美通关关卡 特殊奖励

## 🛠️ 故障排除

### 常见问题

游戏无法启动，提示模块找不到 确保已安装所有依赖：pip install pygame

游戏运行卡顿 尝试降低游戏分辨率或关闭部分特效

音效无法播放 检查sounds文件夹是否存在，或重新下载音效文件

高分无法保存 确保游戏目录有写入权限，检查high_score.json文件

### 技术支持

如遇其他问题，请提交Issue到项目GitHub页面。
